# Maralingeshwar Kattimani's Portfolio

A modern, responsive personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.  
Showcases professional experience, projects, education, certifications, skills, and achievements in a clean, professional layout.

## Features

- **Modern Design:** Clean, professional interface with dark/light mode toggle
- **Responsive:** Works perfectly on desktop, tablet, and mobile devices
- **Side Navigation:** Collapsible navigation with smooth scrolling to sections
- **Easy Customization:** All portfolio information is managed in a single data file
- **Social Links:** Integrated with LinkedIn, GitHub, LeetCode, and CodeChef
- **Comprehensive Sections:** About, Education, Experience, Projects, Skills, Certifications, Competitions, and Connect
- **Dark/Light Mode:** Toggle between themes with persistent preference

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Suezcanaal/Marlingeshwar-portfolio.git
cd hemanth_portfolio
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to see the portfolio.

## Project Structure

```
app/
  page.tsx            # Main portfolio page with all sections
  globals.css         # Global styles
  layout.tsx          # Root layout
  favicon.ico
data/
  portfolioData.ts    # All portfolio data (edit your details here!)
public/
  images/             # Project images and assets
tailwind.config.js    # Tailwind CSS configuration
next.config.ts        # Next.js configuration
tsconfig.json         # TypeScript configuration
```

## Customization

- **Edit your portfolio data:**  
  Open `data/portfolioData.ts` and update all your details:
  - Personal information, contact details
  - Education background (B.E., Class 12, Class 10)
  - Professional experience and internships
  - Projects with GitHub links and live demos
  - Skills categorized by type
  - Certifications with verification links
  - Competition participation and achievements

- **Add project images:**  
  Place project screenshots in the `public/images/` directory

- **Customize styling:**  
  Modify Tailwind CSS classes in `app/page.tsx` for design changes
